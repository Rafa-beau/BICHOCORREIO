# BICHOCORREIO

Você é uma FUMIGA  que vai reprovar, aprovar, aceitar e confiscar provas!!!!!!
Receba moedas e compre habilidades no final de cada rodada

## Controles:

clique direito no refil de tinta para recuperar tinta

Carimbar:
Clique esquerdo -> Aprovar
Clique direito -> Reprovar
